# 📚 Modelo de Classificação Textual para Planos de Ensino

Modelo supervisionado de classificação textual para identificar automaticamente a seção pedagógica de planos de ensino a partir de trechos digitados por usuário, utilizando técnicas de Processamento de Linguagem Natural (PLN).

Projeto desenvolvido como parte de uma Iniciação Científica Júnior.

## 🛠 Tecnologias Utilizadas

- Python
- Pandas
- Scikit-learn
- PyArrow

## 📊 Metodologia

- Divisão dos dados: 60% treino / 40% teste
- Modelos implementados:
  - LinearSVC (85,68% de acurácia)
  - MultinomialNB (78,41% de acurária)

## ▶️ Como executar o modelo?

1. Baixe o arquivo `teaching_plan_text_classifier.ipynb`;
2. Acesse: [Google Colab](https://colab.research.google.com);
3. Clique em **Upload**;
4. Faça upload do notebook;
5. Execute as células na ordem.
